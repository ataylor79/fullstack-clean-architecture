Workout template
    - A workout template can be used multiple times
    - Define level - beginner, intermediate or advanced
    - Define workout type - Strength, cardio, Hiit, yoga, pilates, mobility or hybrid
    - Each workout type can have different attributes (see existsing CreateSet.ts)
    - It must have a main set of exercises
    - Warm up exercises are recommended but not required
    - Cool down exercises are recommended but not required
    - Each main exercise must have a number of sets
    - Each set can have different reps
    - Users can create, delete workout templates
    - Users can only add to workout templates while they are in use, they can not remove exercises from a template if it is in use.

Create a workout plan
    - Using a workout template
    - define which days of the week to apply to the template
    - define how many weeks the plan is for

Logging workouts
    - Log workouts generated by a workout plan
    - log should include when the exercise was completed
    - log should include additional data that should be entered at the time of completing the workout. For example, weight per set or workout notes
    - store date when workout was completed
    - Rate the workout - out of 5.
